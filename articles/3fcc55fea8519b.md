## 前提

- Ruby でアプリケーションを、RSpec でテストを実装していること。
- GitHub Actions で RSpec を実行していること。

## 今回の目的

[SimpleCov](https://github.com/simplecov-ruby/simplecov) で計測した RSpec の coverage を GitHub の Pull Request にコメントしたい。
